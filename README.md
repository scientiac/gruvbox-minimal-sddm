## Help

```
Show help - F1
Cycle select next user - alt+F2
Cycle select next session - alt+F3
Suspend - F10 [fn+F10]
Poweroff - F11 [fn+F11]
Reboot - F12 [fn+F12]
```

> Note:
> Mouse Cursor is hidden and will show up on pressing `F1` for help.
